Embedding Your Resume (Advanced)

Directly embedding PDFs from raw.githubusercontent.com or GitHub repository pages in an <iframe> will typically be blocked by GitHub's security policies (X-Frame-Options, Content-Security-Policy).

If you wish to embed your resume directly on your portfolio page, you have two primary options:

  • Host an HTML version on GitHub Pages: Convert your resume to an HTML file, push it to a GitHub repository, and enable GitHub Pages for that repository. You can then embed the GitHub Pages URL (e.g., https://yourusername.github.io/your-resume-repo/).
  • Self-Host the PDF: Upload the PDF file directly to your own web server where your portfolio is hosted. This avoids cross-origin issues.
